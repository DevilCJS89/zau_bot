# Zau_bot 

## Launching zau_bot

Rviz:

    roslaunch zau_bot_bringup zau_bot_bringup.launch

Rviz with Gazebo:

    roslaunch zau_bot_bringup zau_bot_bringup.launch gui:=true

Teleop:

    roslaunch zau_bot_bringup my_teleop.launch
